Guia Rápido

Como usar o Painel de Análise

  • Filtros: Use os menus de "Modelo", "Cliente" e o seletor de "Período" para refinar os dados exibidos.
  • Navegação por Abas: Alterne entre as abas para diferentes visualizações dos dados.
  • Gráficos Interativos: Clique nas barras ou seções dos gráficos para ver detalhes específicos.
  • Atualizar Dados: O botão busca os dados mais recentes.

Como usar o Registro de Testes

  • Identificação: Seu nome e cor são salvos para agilizar registros futuros. Use o botão "Sair" para trocar de usuário.
  • Seleção Inicial: Comece selecionando o "Modelo do Dispositivo" e preenchendo o "Nome do Cliente".
  • Status dos Componentes: Clique em cada componente para mudar seu status: N/A (Cinza) → OK (Verde) → FALHA (Vermelho).
  • Envio: Após marcar todos os componentes, o botão "Enviar Teste" ficará disponível.
